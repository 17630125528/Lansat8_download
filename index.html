Landsat-download

Automated download of LANDSAT data from USGS website

Download this project as a .zip file Download this project as a tar.gz file

Welcome to LANDSAT-Download pages.

This tool will help you automatically download LANDSAT data form the USGS earthexplorer web site. It was primarily included in CESBIO Multitemp blog, then move here for a better convenience.

There is a short user manual here : https://github.com/olivierhagolle/LANDSAT-Download.wiki.git
